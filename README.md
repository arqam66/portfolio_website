## Deployment

This project is hosted on Firebase. To deploy your own version, follow these steps:

1. Install Firebase CLI:
   ```sh
   npm install -g firebase-tools
   ```
2. Login to Firebase:
   ```sh
   firebase login
   ```
3. Initialize Firebase:
   ```sh
   firebase init
   ```
4. Deploy the project:
   ```sh
   firebase deploy
   ```
